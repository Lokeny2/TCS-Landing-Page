# Tanit Coding School Landing Page

## Overview
A responsive landing page for Tanit Coding School, built with Bootstrap 5. This page showcases the school's mission to provide free coding education.

## Technologies Used
- HTML5
- CSS3 (with custom styles)
- Bootstrap 5.3
- Bootstrap Icons
- Google Fonts (Inter + Playfair Display)

## Features
- Responsive navbar with mobile toggle
- Full-screen hero section with background image
- Stats section (graduates, placement rate, partners)
- Features section with 3 cards
- Simple footer

## What I Learned
- Bootstrap grid system (`container` → `row` → `col`)
- Bootstrap components (navbar, cards, buttons, badges)
- Custom CSS to override Bootstrap defaults
- CSS variables for theming
- Responsive design with `clamp()` and media queries
